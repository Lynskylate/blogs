## 使用最小二乘法求得最优解

上文对于一元线性回归方程使用基础的数学方法可得出结果，当求解多元线性回归方程，为了计算的简便，我们通常会将数据转化为矩阵，通过最小二乘法求出数据的结果。  
我们已经知道损失函数的矩阵形式表示为  
![](/images/6444052/21.png)  
那么我们使用矩阵的求导方法即可得出最优的W  
[Wiki](https://en.wikipedia.org/wiki/Least_squares)已经给出了推导，对于没有梯子的可以参照该博文[**最小二乘法的矩阵形式推导**](http://blog.csdn.net/monsterhoho/article/details/46753673)

通过最小二乘法我们得出最优的w为  
![](/images/6444052/22.png)

```python
# X,Y以上文举例分别为[3,2],[3,1]的矩阵
import numpy as np
X = [[3,1],[4,1],[1,1]]
Y= [[3],[3],[1]
X = np.mat(X)
Y = np.mat(Y)
w = (X.T*X).I*X.T*Y
```
